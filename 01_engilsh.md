# Engilsh

## Object
- Listening: 영어 강의 듣기
- Reading: 공식문서(영어) 읽기
- Speaking: 영어로 정보 전달/설명하기
- Writing: 영어로 문서 작성하기
- 핵심: 기본 문법 + 단어 암기 + 문장 암기

## 기준 점수
| 시험명 | 최소목표점수 | 개인목표점수 |
| :---: | :---: | :---: |
| **토익스피킹** | 140+ | 160+ |
| **토익** | 700+ | 850+ |
| **토익라이팅** | 140+ | 160+ |
| IELTS | 6.5+ | 7.0+ |
- **굵은 글씨** 표시가 있는 것은 최소 필수 조건
