# Embedded

## Object
- C, C++에 대한 깊은 이해
- 문제해결력을 위한 Programmers(Lv3+) 이상의 알고리즘 열량
- 네트워크, 운영체제, 회로이론, 제어공학 등 기본 용어에 대한 이해
- 통신이론 및 회로설계에 대한 이해
- datasheet를 보는 능력 필수

## 기준 점수
| 시험명 | 취득사유 |
| :---: | :---: |
| 리눅스마스터(1급) | 리눅스 기본기 인증 |
| 정보통신기사 | 네트워크, 통신 분야의 기본 이론 인증 |
| 전기기사 | 제어공학, 회로이론 등 기본기 인증 |
- **굵은 글씨** 표시가 있는 것은 최소 필수 조건
